# 🌦 Smart Weather Dashboard

A modern, responsive, and feature-rich weather web application that fetches real-time data using the OpenWeather API and displays it through an interactive dashboard.

---

## 🌐 Live Demo

https://danisasubhani996.github.io/weather-dashboard/

---

## 📂 GitHub Repository

https://github.com/danisasubhani996/weather-dashboard

---

## 🚀 Features

- 🔍 Search weather by city name  
- ⌨️ Press Enter to search instantly  
- 📍 Get weather using current location  
- ⏰ Hourly forecast (next 24 hours)  
- 📅 5-day weather forecast  
- 🌡 Temperature-based dynamic background  
- 🌙 Automatic Day/Night mode  
- 🌧 Weather animations (Rain/Snow effects)  
- 💡 Search suggestions while typing  
- ⏱ Recent search history  
- ⏳ Loading spinner  
- 📱 Fully responsive design  

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- OpenWeather API
